# Gibrid Method to Transmit
## Программный комплекс адаптивной фильтрации телеметрии

Данный проект реализует гибридный метод обработки сигналов телеметрии космических аппаратов. Система автоматически адаптируется к характеру помех, переключаясь между различными алгоритмами фильтрации для минимизации среднеквадратичной ошибки (RMSE).

### 📋 Особенности проекта
* **Гибридность**: Сочетание фильтра Калмана (для гладких участков) и вейвлет-преобразования (для сохранения пиков импульсных воздействий).
* **Адаптивность**: Динамическое переключение каналов обработки на основе оценки мгновенной точности и адаптивного порога.
* **Модульность**: Четкое разделение на генерацию данных, математическую обработку и визуализацию.

### 📂 Архитектура и структура файлов
Проект разделен на логические модули:

1. **`data_gen.py`**: Модуль имитационного моделирования. Генерирует эталонный сигнал, накладывает гауссовский шум и моделирует импульсные воздействия ("удары").
2. **`filters.py`**: Библиотека классических методов. Содержит реализации фильтра Калмана и медианного фильтра.
3. **`wavelet_proc.py`**: Модуль спектральной очистки. Реализует вейвлет-фильтрацию на базе материнского вейвлета `db4`.
4. **`processor.py`**: Интеллектуальное ядро системы. Содержит класс `AdaptiveHybridProcessor`, который управляет логикой смешивания сигналов и весами фильтров.
5. **`test.py`**: Основной сценарий для проведения испытаний. Выполняет расчеты и строит все графики (детальный анализ и массовая статистика).
6. **`main.py`**: Основная задача файла — визуальная верификация. Он берет один эталонный сигнал и показывает в деталях, как ведут себя фильтры в моменты резких изменений (ударов).

### 🚀 Запуск
Для работы требуются Python 3.8+ и следующие библиотеки:
```bash
pip install numpy scipy matplotlib PyWavelets
```

#### Для запуска полного цикла тестирования:
```bash
python test.py
```

#### Для полной демонстрации:
```bash
python main.py
```

Гибридный метод позволяет достичь точности фильтрации с RMSE в пределах 2.5 - 3%, что значительно превосходит показатели одиночных фильтров на нестационарных сигналах телеметрии.
